## 快速使用

### 在线使用

优先推荐大家在线使用，无需编译，访问 <a href="https://rivers.chaitin.cn/tools/" target="_blank">在线使用工具库</a> 即可。

### 自行编译

执行以下命令安装依赖

```
npm install
```

执行以下命令运行开发环境

```
npm run dev
```

执行以下命令打包成静态文件

```
npm run build
```
